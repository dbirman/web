Suffering

I recently watched one of these fun YouTube videos where someone draws pictures and narrates alongside on the topic of Should we kill animals for food? (link at bottom). About halfway through I realized that it was simply listing a number of classic philosophical arguments, setting them up as straw men, and then cutting them down. As if in the end stating that--since we didn't find any reason to eat animals, then there must be no reason. Now certainly better philosophers don't fall into this trap, but I find when reading purely philosophical arguments that they often end up in a hole: they end up being based on our intuitions even if they wrap them up nicely. This is a huge problem: if there is no objective reason to be moral or ethical, then why should we? I think that cognitive neuroscience provides a tentative answer, constrained by a better set of knowledge than philosophical answers have. We'll see how far it gets us.
